Les Variables

Les variables c'est quoi ?

En JavaScript, une variable est un conteneur nommé utilisé pour stocker des données. Ces données peuvent être de différents types, tels que des nombres, des chaînes de caractères, des objets, des tableaux, des fonctions, etc.
Les variables permettent aux développeurs de stocker des valeurs et de les manipuler dans leurs programmes.

Pour déclarer une variable en JavaScript, vous utilisez le mot-clé "var", "let" ou "const", suivi par le nom de la variable.

/!\ Rappel : le mot-clé const ne peut pas être modifié par la suite, vaut mieux le remplacer par le mot-clé "let" ou "var" /!\


Les Booléens

Les booléens c'est quoi ?

En JavaScript, les "booléens" font référence aux valeurs de type "boolean". Un type de données booléen ne peut avoir que deux valeurs : true (vrai) ou false (faux).
Les booléens en JavaScript sont couramment utilisés pour prendre des décisions conditionnelles dans le code, ce qui permet au programme de réagir différemment en fonction de la valeur de vérité d'une expression.

/!\ Rappel : Il faut que la variable contienne "true" ou "false" et ça s'utilise avec "if" "else if" ou "else" la variable peut contenir une valeur "null" qui veut dire absence de valeur ou absence de référence à une valeur (null n'a rien à voir avec undefined) /!\


Les Objets

Les objets c'est quoi ?

En JavaScript, les objets sont des structures de données fondamentales qui permettent de stocker et d'organiser des informations de manière plus complexe que les types de données primitifs tels que les nombres, les chaînes de caractères, les booléens, etc.
Les objets en JavaScript sont des collections de propriétés, où chaque propriété est composée d'une clé (ou nom) et d'une valeur associée. Les objets sont utilisés pour représenter des entités ou des concepts du monde réel.

/!\ Rappel : Les objets sont indiqués avec des {}, on fait une variable et à l'intérieur on mets des renseignements pour éviter de faire plusieurs variables avec leur valeur. Faut le voir comme une fiche pour une personne... ex : let person = {firstname: 'John', lastname:'Doe'}. Pour la commande il faut faire person.firstname /!\
